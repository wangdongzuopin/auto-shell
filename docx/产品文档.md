> 版本：0.1.0 | 更新日期：2026-05-09

## 一、产品定位

**pizz** 是一款面向个人开发者与团队的一站式 AI 工作流平台。

不同于 Cursor / GitHub Copilot 等 IDE 插件模式，pizz 是一个**独立桌面应用**，核心理念是让 AI 成为团队里的多角色协作成员：既能理解项目上下文、执行文件操作，也能从想法、运营、产品、设计、开发、测试到发布形成可流转、可审计、可沉淀的工作闭环。

**个人版一句话定位：** 独立开发者的产品 + 开发 AI 合伙人。

**企业版一句话定位：** 团队内所有职位都能参与想法、评估、设计、开发和交付的一站式 AI 协作平台。

## 二、目标用户

### 个人版核心用户画像

| 维度 | 描述 |
|------|------|
| 角色 | 独立开发者 / 自由职业者 / 单人创业者 |
| 技术栈 | 全栈（前端 + 后端），常用 Vue/React + Node/Java/Go |
| 工作方式 | 独自负责从需求到上线的全流程 |
| 痛点 | 缺少代码审查、缺少产品讨论对象、重复性工作多 |
| 角色边界 | 仅提供产品、开发两个角色 |
| 付费意愿 | 愿意为提升效率的工具付费（¥199-499/年可接受） |

### 企业版核心用户画像

| 维度 | 描述 |
|------|------|
| 团队 | 小型创业团队、外包团队、企业内部工具团队 |
| 职位 | 运营、产品经理、设计、前端、后端、测试、项目负责人 |
| 工作方式 | 多角色围绕同一个项目提出想法、评估、定版、流转、实现 |
| 痛点 | 想法散落、需求流转断层、材料重复产出、知识无法沉淀、AI 使用不可审计 |
| 角色边界 | 支持团队角色与协作角色，多角色参与同一流程节点 |
| 登录要求 | 企业版需要登录，支持成员、权限、审计与企业级数据边界 |

### 次级用户

- 技术爱好者 / 学生
- 产品经理（使用个人版产品角色进行需求分析）
- 需要私有化部署的组织

### 个人版 AI 技能清单（内置 10 套）

1. 编码规范 —— 代码风格、命名、文件组织
2. 安全审查 —— OWASP Top 10、秘密检测
3. TDD 工作流 —— 测试驱动开发方法论
4. 前端模式 —— React/Vue 组件设计、状态管理
5. 架构分析 —— 系统设计、技术选型

7. 原型设计 —— UI/UX 原型指导

9. 竞品分析 —— 市场调研框架
10. 代码审查 —— 质量检查清单

### 企业版角色与流程设想

企业版不再是单一角色切换，而是围绕项目建立“主角色 + 协作角色 + 下一流程角色”的工作方式。

| 角色 | 典型输入 | AI 辅助产物 | 下一流程 |
|------|----------|-------------|----------|
| 运营 | 增长想法、活动方案、用户反馈 | 价值评估、目标人群、转化路径、运营文案 | 产品经理 |
| 产品经理 | 已评估想法、业务目标、用户故事 | PRD、功能边界、流程图、验收标准 | 设计 / 开发 |
| 设计 | PRD、用户路径、页面目标 | HTML 原型、页面结构、交互说明 | 产品定版 / 开发 |
| 开发 | 定版需求、原型、技术约束 | 技术方案、任务拆解、代码修改、Diff | 测试 |
| 测试 | 需求、代码变更、验收标准 | 测试用例、回归清单、缺陷报告 | 发布 |
| 项目负责人 | 项目目标、进度、风险 | 里程碑、风险汇总、跨角色决策记录 | 任意角色 |

推荐 MVP 流程：

```
运营想法 -> AI 可行性评估 -> 产品需求拆解 -> 流程图/HTML 原型 -> 开发任务拆解 -> 代码实现 -> 测试验收 -> 发布材料
```

## 五、商业模式

### 分层策略

| 版本 | 定价 | 目标用户 |
|------|------|----------|
| 开源/个人版 | 免费或低价 Pro | 个人开发者 |
| 企业版 | ¥499-999/年/席位 | 小型团队（2-20 人） |
| 企业定制版 | 按需报价 | 中大型企业 |

### 企业版差异化功能

- 登录与成员体系（企业账号、团队成员、权限）
- 团队协作（共享项目、共享技能、共享知识库）
- 多职位角色体系（运营、产品、设计、开发、测试等）
- 自定义流程流转（想法 -> 评估 -> 定版 -> 实现 -> 验收）
- 产物沉淀（PRD、流程图、HTML 原型、开发计划、测试报告）
- 审计日志（AI 做了什么、谁确认、何时流转）
- 云端数据同步与备份
- 使用量统计与团队管理面板
- SSO / 企业认证
- 优先技术支持
- 私有化部署

### 辅助收入

- 技能市场抽成（15-30%）
- API 代理服务（统一网关赚差价）
