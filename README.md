# Farsight

> 代码的洞察力，开发者的火眼金睛

---

## 命名

Farsight 取意于齐天大圣「火眼金睛」：洞穿虚妄，直抵本质。

---

## 定位

Farsight 是面向专业开发者的桌面级智能 IDE。

不替代思考，只加速洞察；不制造噪音，只放大判断力。

---

## 核心主张

> **看见代码，看见更多**

- **见结构**：跨文件依赖与调用链路，一眼穿透
- **见风险**：潜在 Bug 与性能瓶颈，提前预警
- **见路径**：从需求到实现，最优方案即时推演

---

## 产品特性

| 维度 | 能力 |
| --- | --- |
| 感知 | 全项目语义索引，秒级定位任意符号 |
| 推理 | 上下文感知的代码生成，而非片段拼凑 |
| 共情 | 理解工程意图，延续个人编码风格 |
| 进化 | 随项目成长沉淀记忆模式，越用越默契 |

---

## 设计语言

单色调和，一个圆、一个点、一道光。

剥离色彩干扰，只留形态与光影；在深色 IDE 与浅色系统间保持克制、统一与呼吸感。
